# Farmacia Universal - Tu farmacia online

## Project Overview

Farmacia Universal is an online pharmacy platform designed to provide users with a seamless shopping experience for pharmaceuticals, personal care products, dermocosmetics, maternity items, and nutrition. This web application features a modern, visually appealing interface utilizing Tailwind CSS for styling, ensuring responsiveness and ease of navigation.

## Installation

To run the project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```
2. **Navigate to the project directory**
   ```bash
   cd farmacia-universal
   ```
3. **Open the `index.html` file in your browser**:
   Simply double-click the `index.html` file or right-click and choose to open it with your preferred web browser.

## Usage

Upon opening the application, users can:

- Browse categories of products including medications, personal care, dermocosmetics, baby products, and nutritional items.
- Use the search bar to find specific products.
- View featured products.
- Add products to the cart and simulate a shopping experience.

## Features

- **User-friendly interface**: Easy navigation and browsing categories.
- **Product search functionality**: Allows users to find products quickly.
- **Dynamic cart system**: Users can add products to their cart and see the cart count update in real-time.
- **Responsive design**: The website is optimized for both desktop and mobile devices.
- **Newsletter subscription**: Users can subscribe to receive updates and offers via email.

## Dependencies

This project uses the following external libraries and frameworks:

- **Tailwind CSS**: For styling components.
- **Font Awesome**: For using various icons.
- **Cross-browser compatibility**: Uses scripts to ensure compatibility across different web browsers.

## Project Structure

The project consists of the following files:

- `index.html`: Main HTML file containing the structure of the web application.
- `style.css`: Custom styles for the application that define the appearance of components.
- `script.js`: JavaScript file handling interactivity, including cart functionalities, search debouncing, and newsletter submission.

### Miscellaneous Assets
External resources such as images and icons are loaded from a CDN to enhance the visual appeal and functionality of the application.
